# 📣 Previsão de Churn de Clientes

> Previsão de cancelamento de clientes com ML explicável. Identifica clientes em risco semanas antes e recomenda ações de retenção personalizadas.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/churn-preditivo/

## 📊 Resultados-chave (KPIs)
**0.934** AUC · **+28%** Retenção · **R$ 12M/ano** Receita salva · **4 sem** Antecip.

## 🧠 Tecnologias de IA
Machine Learning · SHAP · Survival Analysis

## ✨ O que faz
- Score de churn por cliente
- Top fatores de risco (SHAP)
- Ação de retenção recomendada
- ROI de campanha de retenção

## 🗄️ Fonte de dados real
[CRM + comportamento](https://www.gov.br/mdic)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com CRM + comportamento
2. **Processamento com IA** — Machine Learning · SHAP · Survival Analysis
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
